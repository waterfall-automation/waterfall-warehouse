# **App Name**: SiccaSync

## Core Features:

- Granular Permission Manager: Role-based access control with JSON-defined permissions stored in a secure Google Sheets backend.
- Visual Storage Navigator: Interactive grid-based layout representing cupboards and storage units with embedded item photo galleries.
- Compliance-Grade Stock Ledger: Automated 30-column inventory system formatted specifically for Indian GST and operational requirements.
- AI Inventory Advisor Tool: Generative AI analysis tool that parses balance levels and location data to suggest optimal restocking priorities and generate brief summaries of recent trends.
- GST Summary Portal: Automated calculation of CGST, SGST, and taxable values across configurable tax brackets (0-28%).
- Centralized Notice Hub: A dynamic board for internal priority-coded notices visible to the entire team based on role permissions.
- Self-Healing App Logic: Google Apps Script automation that periodically validates database health and triggers stock alerts.

## Style Guidelines:

- Primary Color: Professional Midnight Blue (#1B3A6B), selected to evoke stability and trust for organizational data management.
- Background Color: Crisp Porcelain Grey (#F0F3F7), maintaining hue-alignment with the primary blue while ensuring high-clarity data reading.
- Accent Color: Slate Blue (#2D4B8C), an analogous shade chosen for its harmonious relationship with the primary identity, used for UI state changes and interactive cues.
- Headline and Body Font: 'Inter', selected for its geometric clarity and readability in dense data environments. Note: currently only Google Fonts are supported.
- An optimized administrative dashboard featuring a fixed 220px sidebar for efficient multitasking and breadcrumb navigation on every page.
- Functional transition micro-interactions applied to state-change notifications and slide-over panels for item inspection.
- Line-based, weight-consistent iconography from a set emphasizing inventory tools, ledger books, and facility locations.
